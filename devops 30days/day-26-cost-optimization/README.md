# Day 26 - Cost Optimization

## Target

Reduce cloud waste without hurting reliability.

## Learn Deeply

- Rightsizing.
- Autoscaling.
- Idle resources.
- Storage lifecycle.
- NAT/data transfer cost.
- Log retention.
- Savings plans/committed use.
- Tags/labels.

## Hands-On Lab

Create a cost report plan:

- Top services by cost.
- Untagged resources.
- Idle disks/IPs/load balancers.
- Non-prod scheduling.
- Storage lifecycle.
- Logging retention.

## Interview Angle

Say:

```text
I optimize cost by measuring first, removing waste second, and protecting reliability throughout.
```

## AWS/GCP Mapping

AWS Cost Explorer/Budgets map to GCP Billing reports/budgets. Labels in GCP map to tags in AWS.

## Daily Motivation

Saving cost while maintaining reliability is a strong engineering signal.
