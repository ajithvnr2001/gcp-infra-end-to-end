## 🗓️ Weekly Breakdown

- **Week 1** (Day 1-7): Core Programming - variables, conditions, loops, functions, arrays, dicts
- **Week 2** (Day 8-14): Linux Automation - files, grep, awk, sed, find, processes
- **Week 3** (Day 15-21): Real DevOps Scenarios - disk, services, ports, logs, backup, health
- **Week 4** (Day 22-30): Python DevOps - JSON, APIs, subprocess, final project

## 🎯 Goal
"I should be able to look at a DevOps problem and automate it using Bash or Python."

## 📖 How to Use
1. Each day folder contains a `README.md` with motivation, learning objectives, examples, and exercises
2. Study both Bash and Python for each topic
3. Code along with every example
4. Complete all exercises before moving to next day
5. Day 7 and Day 14 are revision/project days
