# Day 22 - AWS Monitoring And Audit

## Target

Operate AWS systems using telemetry and audit logs.

## Learn Deeply

- CloudWatch Logs.
- CloudWatch Metrics.
- CloudWatch Alarms.
- CloudTrail.
- VPC Flow Logs.
- ALB access logs.
- RDS enhanced monitoring.
- AWS Budgets.

## Hands-On Lab

Design monitoring for a 3-tier AWS app:

- ALB 5xx and latency.
- ECS/EKS CPU/memory/restarts.
- RDS connections/CPU/storage.
- CloudTrail IAM/security group changes.
- Budget alert.

## Interview Angle

Say:

```text
CloudWatch tells me service health. CloudTrail tells me who changed what.
```

## AWS/GCP Mapping

CloudWatch is like Cloud Monitoring/Logging. CloudTrail is like Cloud Audit Logs.

## Daily Motivation

The fastest way to debug AWS is knowing where each signal lives.
