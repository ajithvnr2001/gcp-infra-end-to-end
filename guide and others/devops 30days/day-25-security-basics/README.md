# Day 25 - Security Basics For DevOps

## Target

Answer practical security questions from a DevOps role perspective.

## Learn Deeply

- Least privilege.
- Private networking.
- Secrets management.
- Encryption at rest/in transit.
- Image scanning.
- RBAC.
- Audit logs.
- Patch management.

## Hands-On Lab

Create two checklists:

1. Secure Kubernetes deployment.
2. Secure AWS VPC/application deployment.

Include identity, network, runtime, secrets, logs, and scanning.

## Interview Angle

Say:

```text
I secure systems by default: least privilege IAM, private networking, managed secrets, encryption, audit logs, and automated scanning.
```

## AWS/GCP Mapping

AWS WAF/Shield maps to Cloud Armor. Secrets Manager maps to Secret Manager. CloudTrail maps to Audit Logs.

## Daily Motivation

Security is not separate from DevOps. It is part of every production decision.
