# Day 11 Project + AWS Focus

## Project Connection

Terraform provisions modules for VPC, GKE, and Cloud SQL.

## GCP To AWS Mapping

AWS Terraform equivalent:

```text
VPC module -> AWS VPC, subnets, route tables, NAT Gateway
GKE module -> EKS module
Cloud SQL module -> RDS module
```

## Project Question

What should you say if asked why Terraform?

Answer:

```text
Terraform makes cloud infrastructure versioned, repeatable, reviewable, and safer to rebuild. It also helps detect drift.
```
